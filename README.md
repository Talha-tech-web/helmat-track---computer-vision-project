# 🪖 Helmet Detection System using YOLOv8

A computer vision project that detects whether a person is wearing a helmet or not in real-time road scenarios using YOLOv8.

---

## 🚀 Project Overview

This project focuses on improving road safety by automatically detecting helmet usage among riders. The model classifies individuals into:

- ✅ With Helmet  
- ❌ Without Helmet  

It is designed for real-world applications such as traffic monitoring, law enforcement, and smart city systems.

---

## 📊 Dataset

- 📁 Custom dataset with **3000+ images**
- 🏍️ Real-world road and traffic scenarios
- 🏷️ Annotated using bounding boxes for helmet detection

---

## ⚙️ Model Training

- 🧠 Model: YOLOv8 (Ultralytics)
- ⏱️ Training Time: ~2 days
- 💻 Platform: Google Colab
- ⚡ Used multiple Colab sessions due to limited GPU availability

---

## 🧩 Challenges Faced

- Limited GPU resources
- Managing long training sessions
- Ensuring dataset quality and consistency
- Optimizing model performance under constraints

---

## 🎯 Results

- Successfully detects helmet vs no-helmet cases
- Works on both images and video streams
- Real-time detection with bounding boxes and confidence scores

---

## 🛠️ Tech Stack
